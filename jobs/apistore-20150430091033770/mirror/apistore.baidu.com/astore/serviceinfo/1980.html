接口地址：http://apistore.baidu.com/microservice/aqi

请求方法：GET

请求参数：
参数名 类型 必填 参数位置 描述 默认值
city string 是 urlParam 城市名，具体支持城市请查看基本介绍 北京

http://apistore.baidu.com/microservice/aqi?city=北京

    aqi: 38,  //空气质量指数
    level: "优", //空气等级
    core: "" //首要污染物
    }
}
