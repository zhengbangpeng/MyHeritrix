服务简介

南京飞拓是国内专业的短信接口平台服务商，致力于降低企业移动增值服务门槛。于中国移动、中国联通、中国电信保持长期深入的合作关系。坚持产品自主研发，坚持7*24小时全天服务。短信接口，短信平台，短信验证码接口。

特色功能

1、短信验证码接口/订单通知短信专用接口
南京飞拓短信验证码接口/订单通知短信专用接口，具有全国全网发送、3-5秒内响应、100%到达、通道稳定免维护等优势。专为网站短信验证码、网站订单通知等互动应用开设。
    
2、短信验证码接口应用领域
网站手机验证、APP应用手机验证、订单通知、物流提醒、触发类短信。

3、短信验证码接口特点
适用于同电子商务网站、企业内部管理系统进行整合；可应用于手机验证、订单通知、快递跟踪、会员提醒等。

API列表

短信发送接口API

接口地址：http://222.185.228.25:8000/msm/sdk/http/sendsms.jsp

请求方法：POST

请求参数：
参数名 类型 必填 描述
username string 是 账号
scode string 是 密码
mobile string 是 手机号码
content string 是 短信内容
extcode string 否 扩展子号
sendtime string 否 发送时间
signtag string 否 签名
tempid string 否 模板编号
msgid string 否 消息id
请求示例： 
http://222.185.228.25:8000/msm/sdk/http/sendsms.jsp?username=NTY000000&scode=123456&mobile=13805100000&content=你好101540
JSON返回示例： 
{
    code : 0#数字#数字
}
XML返回示例： 
<result>
    <code>0#数字#数字</code>
</result>

备注：0#数字#数字 格式：返回值#提交计费条数#提交成功号码数 100 发送失败 101 用户账号不存在或密码错误 102 账号已禁用 103 参数不正确 105 短信内容超过500字、或为空、或内容编码格式不正确 106 手机号码超过100个或合法的手机号码为空 108 余额不足 109 指定访问ip地址错误 110#(敏感词A,敏感词B) 短信内容存在系统保留关键词，如有多个词，使用逗号分隔：110#(李老师,XX,成人) 114 模板短信序号不存在 115 短信签名标签序号不存在
