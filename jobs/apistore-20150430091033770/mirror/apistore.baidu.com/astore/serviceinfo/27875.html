API列表

获取某公司电子券列表

接口地址：http://qr.hljbt.com/GetQR_one.aspx

请求方法：GET

请求参数：
参数名 类型 必填 描述
company_id string 是 公司唯一ID
请求示例： 
http://qr.hljbt.com/GetQR_one.aspx?company_id=52
JSON返回示例： 
{"flag":"true","QRList":[{"company_id":"52","QRtype":"TiYan","price":"0","state":"F","linkphone":"0451-86282777","address":"南岗区花园街403号，新世界百货5楼","company":"悦四季品质海鲜自助火锅","remark":"凭电子券消费大厅午餐原价148元，现价125元。晚餐原价168元，现价145元。包房午餐原价198元，现价175元，晚餐原价218元，现价195元。（不可与店内其他优惠同时使用）
休息时间年三十(2月18日）","EndDate":"2015-03-10"}]}

备注：取到可用优惠信息，可通过获取电子码API，获取二维码。

获取电子码

接口地址：http://qr.hljbt.com/GetQRCode.aspx

请求方法：GET

请求参数：
参数名 类型 必填 描述
openid string 是 用户ID
company_id string 是 公司ID
price string 是 优惠的金额
BackUrl string 是 扫描二维码转到的URL
username string 是 领取人
userphone string 是 邻取人手机号
UserFrom string 否 方便统计来自哪个合作渠道，可为空
请求示例： 
http://qr.hljbt.com/GetQRCode.aspx?company_id=52&price=0&openid=userID&username=sadsfar&UserFrom=HLJSL&userphone=4154156&BackUrl=http://wap.88882222.cn/zhuanti/jiekou/huodong/jiekou_merchants.php
JSON返回示例： 
{"flag":"true","QR_id":"322221","QRUrl":"codes/YSJ201502136931.png"}

备注：flag=true时，获取电子码成功 QRURL为二维码图片地址，请在前面加上接口地址访问：http://qr.hljbt.com/

验证消费

接口地址：http://qr.hljbt.com/cost.aspx

请求方法：GET

请求参数：
参数名 类型 必填 描述
QR_id string 是 电子码ID
PWD string 是 商户密码
Useopenid string 是 操作人员ID
请求示例： 
http://qr.hljbt.com/cost.aspx?qr_id=322221&pwd=123&useropenid=123
JSON返回示例： 
{"flag":"true","msg":"验证成功！"}
