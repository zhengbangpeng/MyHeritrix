去调试 >> 电视节目时刻表查询

接口地址：http://api.46644.com/tv/query/

请求方法：GET

请求参数：
参数名 类型 必填 参数位置 描述 默认值
tvid int 是 urlParam 电视的频道ID，对应节目频道里面的ID 464
starttime int 否 urlParam 开始时间 Unix时间戳 1367197800 1367197800
endtime int 否 urlParam 结束时间 Unix时间戳 1418379395（每次返回最多500条记录） 1418399395
appkey string 是 urlParam 你申请的appkey 1307ee261de8bbcf83830de89caae73f
请求示例： 
http://api.46644.com/tv/query/?tvid=464&starttime=1367197800&endtime=1418399395&appkey=1307ee261de8bbcf83830de89caae73f
