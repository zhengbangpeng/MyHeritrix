服务简介

【千橙】是国内专业从事短信接口、行业通知短信、会员营销短信服务，支持多语言多平台。专业结合三大运营商优质通道，每秒速度可达200条，极速稳定，可上行，可扩展。支持最新虚拟号段“170”.
【千橙】短信平台提供免费短信接口，多用于互联网网站短信验证，如：网站注册、身份确认以及重要资料或资金安全的修改与变动通知。
行业通知短信多用于服务行业或政企内部员工的消息通知，如：【充值通知】、【消费提醒】、【故障提醒】、【订单通知】、【物流通知】、【验证码】等。

会员营销短信多用于各行业新品上市、特色推广、商超活动，节日祝福等。
    
    国内专业的短信接口平台服务商，致力于降低企业移动增值服务门槛。于中国移动、中国联通、中国电信保持长期深入的合作关系。坚持产品自主研发，坚持7*24小时全天服务。短信接口，短信平台，短信验证码接口。

1、短信验证码接口/订单通知短信专用接口
中舜短信验证码接口/订单通知短信专用接口，具有全国全网发送、3-5秒内响应、100%到达、通道稳定免维护等优势。专为网站短信验证码、网站订单通知等互动应用开设。

API列表

千橙短信接口API

接口地址：http://112.124.17.46:7001/send

请求方法：GET

请求参数：
参数名 类型 必填 描述
username string 是 用户名
password string 是 密码
mobile string 是 手机号
content string 是 发送内容
请求示例： 
http://112.124.17.46:7001/send?username=user&password=888888&mobile=13900000000&content=短信内容【千橙科技】​
get post方式都支持，
接口开发最好使用post，测试可以使用get方式测试。
JSON返回示例： 
1,2015031702348765
XML返回示例： 
1,2015031702348765

备注：返回值 描 述 -1 用户名或者密码不正确 1,发送ID 发送成功 2 余额不足 3 扣费失败 6 有效号码为空 7 短信内容为空
