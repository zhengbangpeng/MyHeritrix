服务简介

将外语翻译成中文谐音。目前支持英语，日语和韩语三种语言。
"Thank you" => "三可 有"；
"こんにちは" => "考恩你起瓦";
"안녕하세요" => "阿宁哈些要".

特色功能

1. 不用掌握任何外语知识，与外国友人谈笑风生！
2. 任何时候，脱口而出！
3. 语言游戏，出境医疗协助，旅游，国际贸易，卡拉OK, 多种应用场景，任您探索！

使用帮助

用户登录我们的官网，简单注册后获得user_key， 即可免费调用。（免费额度 100次每天）。

API列表

去调试 >> 外语立刻说-英文谐音

接口地址：http://howtospeak.org:443/api/e2c

请求方法：GET

请求参数：
参数名 类型 必填 参数位置 描述 默认值
notrans int 否 urlParam 输入文本是否为外语 0
text string 是 urlParam 输入文本 你好
user_key string 是 urlParam user_key dfcacb6404295f9ed9e430f67b641a8e
请求示例： 
http://howtospeak.org:443/api/e2c?user_key=dfcacb6404295f9ed9e430f67b641a8e &notrans=0&text=%E4%BD%A0%E5%A5%BD
JSON返回示例： 
{"chinglish": "和楼", "english": "Hello"}

备注：调用API时，请使用您在我们官网注册后得到的user_key。
