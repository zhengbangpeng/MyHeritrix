服务简介

川道短信验证码接口/川道订单通知短信专用接口
106短信接口网无线短信验证码接口/订单通知短信专用接口，具有全国全网发送、3-5秒内响应、100%到达、通道稳定免维护等优势专为网站短信验证码、网站订单通知等应用开设。

特色功能

适用于同电子商务网站、APP开发、企业内部管理系统进行整合；
可应用于手机验证、订单通知、快递跟踪、会员提醒等
支持移动、联通、电信号码的发送	
正规106网关通道
响应速度快，5-8秒可到达	
上下行一致：用户可以回复短信上行
支持300个字的长短信
自定义签名、自定义发送模板	
接口采用http和webservice方式
完善的接口程序，接入非常简单	
支持PHP、JSP、ASP等所有程序

使用帮助

开发者必读：
1，内容格式：内容+【企业签名】
2，同号码同内容同时段不超过2条，同号码不同内容同时段不能超过3条。
3，技术支持：021-33199099-0

API列表

去调试 >> 川道短信接口

接口地址：http://sms.shsixun.com/Port/Msg.asmx

请求方法：GET

请求参数：
参数名 类型 必填 参数位置 描述 默认值
username string 是 urlParam 系统分配的用户帐号
password string 是 urlParam 系统分配的用户密码
phonelist string 是 urlParam 下发信息的手机号码，多个手机号用半角逗号分隔，每个提交包控制在100个手机号码内。
msg string 是 urlParam 下发信息的内容，70个字，超长系统自动截取。
SendDatetime string 是 urlParam 发送时间，空为立即发送
请求示例： 
http://sms.shsixun.com/Port/default.ashx?method=SendSms&username=test&password =001&phonelist=13472510008&msg=测试&SendDatetime=
JSON返回示例： 
状态值(字符串)	说明
DELIVRD	短消息转发成功
EXPIRED	短消息超过有效期
UNDELIV	短消息是不可达的
UNKNOWN	未知短消息状态
REJECTD	短消息被短信中心拒绝
DTBLACK	目的号码是黑名单号码
ERR:104	系统忙
其他	网关内部状态

备注：1）、通讯协议是HTTP（POST方式提交内容）。 2）、请求应答过程中使用的中文编码格式为UTF-8。 3)、应答为状态码，每个接口下定义具体的状态码
