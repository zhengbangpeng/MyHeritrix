服务简介

      第一信息短信是一家从事电信增值业务的公司，旗下有上海第翼信息科技有限公司、上海博勇信息科技有限公司、上海笃信信息科技有限公司，提供精准的短信群发服务，拥有全国领先的技术团队，为企业打造更可靠的通信平台.
       第一信息是中国移动、中国联通、中国电信等中国主要电信运营商的资深战略合作伙伴，在移动商务的平台技术、应用产品、销售渠道和服务网络等领域与其开展了全方位的深度合作，并积累了丰富的合作经验。第一信息先后获得了由工信部颁发的“中华人民共和国增值电信业务经营许可证”、“中华人民共和国电信与信息服务业务经营许可证”和由工信部批准的“全国三网合一短消息类服务第三方短信平台接入代码使用证书”。
      接口采用云端处理技术，支持各种开发语言，PHP，java，c#，IOS，安卓等等，采用http POST的接口形式，使接入更加便捷，电商，ERP不二之选

特色功能

1、支持短信报告、短信上行的推送报告

API列表

第一信息短信平台接口

接口地址：http://sms.1xinxi.cn/asmx/smsservice.aspx

请求方法：POST

请求参数：
参数名 类型 必填 描述
name string 是 用户账号
pwd string 是 web平台：基本资料中的接口密码
content string 是 发送内容（1-500 个汉字）UTF-8编码
mobile string 是 手机号码。多个以英文逗号隔开
stime string 否 可选参数。发送时间，填写时已填写的时间发送，不填时为当前时间发送
sign string 是 用户签名。
type string 是 固定值 pt
extno string 是 扩展码
请求示例： 
http://sms.1xinxi.cn/asmx/smsservice.aspx?name=test&pwd=112345&content=testmsg&mobile=13566677777,18655555555&stime=2012-08-01 8:20:23&sign=testsign&type=pt&extno=123
JSON返回示例： 
0, 20130821110353234137876543,0,500,0,提交成功
