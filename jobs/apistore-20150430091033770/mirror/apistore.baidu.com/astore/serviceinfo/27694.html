去调试 >> 免费短信接口

接口地址：http://web.duanxinwang.cc/asmx/smsservice.aspx

请求方法：POST

请求参数：
参数名 类型 必填 参数位置 描述 默认值
name string 是 urlParam 必填参数。用户账号 test
pwd string 是 urlParam 必填参数。（web平台：基本资料中的接口密码） 112345
content string 是 urlParam 必填参数。发送内容UTF-8编码，建议每次1000个 格式：内容#@#号码#@@#内容#@#号码 例：你好张三#@#13566666666#@@#你好李四#@#13999999999 testmsg
mobile string 是 urlParam 用户手机号码 13566677777
stime string 否 urlParam 可选参数。发送时间，填写时已填写的时间发送，不填时为当前时间发送 2012-08-01 8:20:23
sign string 是 urlParam 可选参数。用户签名。 testsign
type string 是 urlParam 必填参数。固定值 gx pt
extno string 是 urlParam 可选参数，扩展码，用户定义扩展码，只能为数字 123
请求示例： 
http://web.duanxinwang.cc/asmx/smsservice.aspx?name=dxw&pwd=1C8E70B4C8EB3E3700FC70ED43E7&content=testmsg&mobile=18255183002&stime=2015-05-01 8:00:00&sign=testsign&type=pt&extno=123
JSON返回示例： 
0,2015050108000000029787155,0,1,0,提交成功
