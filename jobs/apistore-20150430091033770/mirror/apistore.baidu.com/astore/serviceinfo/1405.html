服务简介

高德地图 JavaScript API，是由 JavaScript 语言编写的应用程序接口，它能够帮助您在网站中构建功能丰富、交互性强的地图应用程序。JavaScript 地图 API 不仅包含构建地图的基本接口，还提供了诸如本地搜索、路线规划等数据服务，您可以根据自己的需要进行选择性的使用。
高德地图移动端API是一套简单的地图应用开发调用接口，可支持地图显示与操作，兴趣点搜索，地理编码、离线地图等功能。开发者可以轻松地将其嵌入自己的应用程序，并使用相关的地图功能。
