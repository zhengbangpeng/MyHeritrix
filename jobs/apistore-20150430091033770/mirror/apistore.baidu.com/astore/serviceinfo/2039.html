服务简介

宅言API通过调用ACman用户分享和系统收录的经典台词数据库，为网站提供有趣的句子分享。

特色功能

1.收录动画、漫画、小说和网络中让用户有共鸣感的句子，随机展现。
2.简单易用，您只需通过几行代码就可以调用宅言API。
3.数据强大，目前已收录近1000条动漫台词，每天都会增加。

使用帮助

API列表

去调试 >> 宅言API（ACman）

接口地址：http://zhaiyan.acman.cn/api

请求方法：GET

请求参数：
参数名 类型 必填 参数位置 描述 默认值
id string 否 urlParam 返回指定ID的一句话，如果为空则返回任意ID的宅言。 73
请求示例： 
http://zhaiyan.acman.cn/api?id=73
JSON返回示例： 
{
    "id":"73",
     "zhaiyan":"请不要忘记，那给你带来感动的，名为二次元的理想乡。",
     "cat":"e",
     "catname":"原创",
     "author":"",
     "show":null,
     "source":"原创",
     "date":"2014-10-30",
     "img":null
}

备注：前面会出现一个看不见的的字符，请您删除掉后再解析，原因是我们在处理数据的时候编码BOM头的问题
