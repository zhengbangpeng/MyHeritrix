服务简介

翻译API是面向开发者推出的免费翻译服务开放接口，任何第三方应用或网站都可以通过使用翻译API为用户提供实时优质的多语言翻译服务，提升产品体验。
1、翻译API
支持多种语言互译：覆盖中文、英语、日语、韩语、西班牙语、法语、泰语、阿拉伯语、俄语、葡萄牙语、德语、意大利语等热门语种，新增粤语和普通话，文言文与白话文互译；
2、词典API：
支持语种：中文、英文
输入单词返回结果包括：英式音标、美式音标、词性、详细释义（多个结果）

特色功能

1、免费
2、服务稳定
3、支持语种多，包含特色翻译（粤语、文言文）
4、提供翻译和词典两种API

去调试 >> 翻译

接口地址：http://apistore.baidu.com/microservice/translate

请求方法：GET

请求参数：
参数名 类型 必填 参数位置 描述 默认值
query string 是 urlParam 待翻译内容,UTF-8，urlencode编码 hello
from string 是 urlParam 源语言语种：语言代码或auto en
to string 是 urlParam 目标语言语种：语言代码或auto zh
请求示例： 
http://apistore.baidu.com/microservice/translate?query=hello&from=en&to=zh
JSON返回示例： 
{
    "errNum": 0,
    "errMsg": "success",
    "retData": {
        "from": "en",
        "to": "auto",
        "trans_result": [
            {
                "src": "hello",
                "dst": "您好"
            }
        ]
    }
}
