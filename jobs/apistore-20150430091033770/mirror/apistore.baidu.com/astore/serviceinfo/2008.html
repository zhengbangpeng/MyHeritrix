服务简介

图灵机器人是国内第一家个性化智能机器人开放平台，为广大开发者提供免费的智能机器人API端口。图灵机器人基于DeepQA深度问答技术，对中文的识别准确率高达90%，是目前中文语境下智能度最高的机器人。图灵机器人具有完全免费、高智能、一键接入、个性化、跨平台五大特点，通过图灵机器人开放平台，任何人都可以方便快捷的为微信公众号、微博、QQ群、WEB网站、智能客服系统以及智能家居系统、智能车载系统等软硬件领域接入一位聪明的图灵机器人。

1）自然语言聊天，像真人一样对答如流；

5）机器人调教，你的风格自然由你做主；
6）调用次数不够？需要多少统统给你。

API列表

图灵机器人

接口地址：http://www.tuling123.com/openapi/api

请求方法：GET

请求参数：
参数名 类型 必填 描述
key string 是 开发者进行注册后获取
info string 是 如“你好”、“北京天气”等请求的内容；文字编码采用utf-8格式
userid string 是 用户id，自己定义即可
请求示例： 
http://www.tuling123.com/openapi/api?key=KEY&info=你漂亮么
JSON返回示例： 
{

"code":100000,

"text":"恩恩，害羞ing……"

}

备注：示例为文字类请求及返回结果的示例，图片类、链接类、新闻类、地理位置类等因版面限制无法统一上传，可前往图灵机器人官方网站进行查询。

图灵机器人

接口地址：http://www.tuling123.com/openapi/api

请求方法：GET

请求参数：
参数名 类型 必填 描述
key string 是 开发者注册后免费获取
info string 是 如发送“你好”、“北京热吗”、“附近酒店”等内容

{

"code":309000,

"text":"********",

"list":[{

"name":"",

"price":"",

"satisfaction":"",

"count":"",

"detailurl":""

"icon":""

}]

}
